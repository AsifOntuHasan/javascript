# JavaScript Programming Examples

![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

A comprehensive collection of **JavaScript** programming examples covering fundamentals, algorithms, data structures, and more. Ideal for beginners and intermediate developers mastering JavaScript.

## 📋 Overview

This repository contains **100+ individual JavaScript programs** organized by difficulty and topic. Each file demonstrates a specific concept or problem-solving approach using vanilla JavaScript.

### Repository Structure
- All programs are standalone `.js` files
- Numbered sequentially (e.g., `001_01_01_Hello_World_Program.js`)
- Progressive learning path from basics to advanced topics

## 🚀 Features

- Pure vanilla JavaScript (no frameworks)
- Console-based examples (many runnable in browser too)
- Clean, educational code with clear logic
- Great for learning, practice, and interview preparation

## 📚 Program Categories

### Level 1: Fundamentals
- Hello World
- Simple Calculator
- Even/Odd Checker
- Basic Math Operations
- File Handling Concepts

### Level 2-3: Control Flow & Functions
- Palindrome Check
- Fibonacci Sequence
- Prime Numbers
- String Manipulations
- Multiplication Tables

### Level 4-5: Intermediate Topics
- Arrays & Loops
- Basic OOP (Classes & Objects)
- Sorting Algorithms (Bubble, Selection, Quick Sort, etc.)
- Recursion
- Pattern Generation

### Level 6+: Advanced
- Data Structures (Stack, Queue, Binary Search Tree)
- Mathematical & Statistical Algorithms
- Random Data Generation
- Number System Conversions
- Advanced Algorithms

## 🛠️ How to Run

### Using Node.js (Recommended)
```bash
node filename.js
```

### In Browser
1. Create an `index.html` file:
   ```html
   <script src="filename.js"></script>
   ```
2. Open the HTML file in your browser and check the console (F12).

### Example:
```bash
node 001_01_01_Hello_World_Program.js
```

## 📖 Learning Path

1. Start with `001_01_01_Hello_World_Program.js`
2. Follow files in numerical order
3. Modify and experiment with the code
4. Combine examples to build small projects
